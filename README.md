# Калькулятор микронутриентов — GitHub Pages

Статическая версия калькулятора поступления микро- и макронутриентов на низкокалорийной диете с поддержкой OPTIFAST. Логика полностью повторяет Streamlit-приложение из корня репозитория.

## Структура

```
github-pages/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
└── data/
    ├── menu_optimized.json
    └── Optifast.json
```

## Локальный просмотр

Из папки `github-pages` запустите простой HTTP-сервер (нужен для загрузки JSON):

```bash
# Python
python -m http.server 8080

# или Node.js
npx serve .
```

Откройте в браузере: http://localhost:8080

## Публикация на GitHub Pages

### Вариант 1 — папка `/docs` или `/github-pages` в репозитории

1. Загрузите содержимое папки `github-pages` в репозиторий GitHub.
2. В настройках репозитория: **Settings → Pages**.
3. В **Source** выберите ветку `main` и папку с файлами (`/github-pages` или `/docs`).
4. Сохраните — сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

### Вариант 2 — отдельный репозиторий `username.github.io`

Скопируйте содержимое `github-pages` в корень такого репозитория — сайт откроется по адресу `https://<username>.github.io/`.

## Возможности

- Выбор пола, калорийности (1200–2000 ккал) и типа меню
- Отображение суточного меню по приёмам пищи
- Расчёт макро- и микронутриентов с цветовой индикацией
- Переключатель «Диета / Диета + 1 саше OPTIFAST» с пересчётом показателей
- Холестерин не отображается (как в актуальной версии Streamlit-приложения)

## Обновление данных

При изменении меню или состава OPTIFAST обновите файлы в `data/` и перезалейте на GitHub.
